## FAQs

### How can Flotilla be completely free?

This is an **academic project** built with engineering rigor, not a venture-backed startup optimizing for revenue. We follow the open-source philosophy: build in public, share knowledge, improve together.

### Will there be paid features in the future?

**Never.** All current and future features will remain open source under MIT license. We may offer optional paid services (like managed hosting), but the software itself is free forever.

### How do I contribute?

Visit our [GitHub repository](https://github.com/CPU-JIA/Cloud-Dev-Platform), read the docs, submit issues, or contribute code. Every contribution helps build consensus!

### What infrastructure do I need to self-host?

Minimum requirements:
- **CPU**: 2 cores
- **RAM**: 4GB
- **Storage**: 20GB
- **Software**: Docker + Docker Compose

See our [Quick Start Guide](/docs/getting-started) for detailed setup instructions.
